Jellyfin2HQPlayer Plugin - macOS Deployment Guide

--------------------------------------------------
1. Files
--------------------------------------------------
Place both files in the same directory:

deploy-jellyfin-plugin-macos-arm64.sh
Jellyfin2HQPlayerPlugin-1.0.0.tar.gz


--------------------------------------------------
2. Install
--------------------------------------------------
chmod +x deploy-jellyfin-plugin-macos-arm64.sh
./deploy-jellyfin-plugin-macos-arm64.sh


--------------------------------------------------
3. What the script does
--------------------------------------------------
The script will automatically:

- Extract the package
- Install plugin files to Jellyfin plugin directory
- Clean old plugin files
- Fix file permissions


--------------------------------------------------
4. Restart Jellyfin (IMPORTANT)
--------------------------------------------------
You MUST restart Jellyfin manually:

- Quit Jellyfin.app
- Open Jellyfin.app again

Plugin will NOT take effect without restart


--------------------------------------------------
5. Verify
--------------------------------------------------
Open Jellyfin in browser:

http://<IP>:8096

Go to:

Dashboard -> Plugins

You should see:

Jellyfin2HQPlayer


--------------------------------------------------
6. Notes
--------------------------------------------------
- Run in the same directory as:
  deploy-jellyfin-plugin-macos-arm64.sh
  Jellyfin2HQPlayerPlugin-1.0.0.tar.gz

- The script automatically detects the latest tar.gz package

- No sudo required


--------------------------------------------------
Summary
--------------------------------------------------
Two files + one command + restart = done
