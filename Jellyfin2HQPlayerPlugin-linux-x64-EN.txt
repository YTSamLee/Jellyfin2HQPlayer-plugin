Jellyfin2HQPlayer Plugin - Linux x64 Deployment Guide

--------------------------------------------------
1. Files
--------------------------------------------------
Place both files in the same directory:

deploy-jellyfin-plugin-linux-x64.sh
Jellyfin2HQPlayerPlugin-1.0.0.tar.gz


--------------------------------------------------
2. Install
--------------------------------------------------
chmod +x deploy-jellyfin-plugin-linux-x64.sh
./deploy-jellyfin-plugin-linux-x64.sh


--------------------------------------------------
3. What the script does
--------------------------------------------------
The script will automatically:

- Extract the package
- Install plugin files to Jellyfin plugin directory
- Fix file permissions
- Restart Jellyfin


--------------------------------------------------
4. Verify
--------------------------------------------------
Open Jellyfin in browser:

http://<IP>:8096

Go to:

Dashboard -> Plugins

You should see:

Jellyfin2HQPlayer


--------------------------------------------------
5. Notes
--------------------------------------------------
- Run in the same directory as:
  deploy-jellyfin-plugin-linux-x64.sh
  Jellyfin2HQPlayerPlugin-1.0.0.tar.gz

- Do not rename the package file unless you also update the script

- sudo is handled inside the script where needed


--------------------------------------------------
Summary
--------------------------------------------------
Two files + one command = done
