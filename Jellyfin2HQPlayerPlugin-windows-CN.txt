Jellyfin2HQPlayer 插件 - Windows x64 部署指南

--------------------------------------------------
1. 文件准备
--------------------------------------------------
将以下两个文件放在同一目录：

deploy-jellyfin-plugin-windows-x64.ps1
Jellyfin2HQPlayerPlugin-1.0.0.tar.gz


--------------------------------------------------
2. 打开 PowerShell
--------------------------------------------------
在当前目录右键 → 打开终端（PowerShell）

或手动进入目录


--------------------------------------------------
3. 临时允许脚本执行
--------------------------------------------------
执行：

Set-ExecutionPolicy -Scope Process Bypass

说明：
仅对当前 PowerShell 会话生效，关闭窗口后自动恢复


--------------------------------------------------
4. 安装
--------------------------------------------------
执行：

.\deploy-jellyfin-plugin-windows-x64.ps1


--------------------------------------------------
5. 重启 Jellyfin（必须）
--------------------------------------------------
必须手动重启 Jellyfin，否则插件不会生效

托盘版本（推荐）：
- 右下角托盘 → Stop Jellyfin
- 再 Start Jellyfin

服务版本：
- 在“服务”中重启 Jellyfin
或执行：

  Restart-Service Jellyfin


--------------------------------------------------
6. 验证
--------------------------------------------------
浏览器打开：

http://<IP>:8096

进入：

Dashboard → Plugins（控制台 → 插件）

应看到：

Jellyfin2HQPlayer


--------------------------------------------------
7. 注意事项
--------------------------------------------------
- 请在以下文件所在目录执行：

  deploy-jellyfin-plugin-windows-x64.ps1
  Jellyfin2HQPlayerPlugin-1.0.0.tar.gz

- 如修改插件包文件名，需要同步修改脚本

- 安装过程中必须确保 Jellyfin 未运行


--------------------------------------------------
总结
--------------------------------------------------
两个文件 + 一条命令 + 重启 = 完成部署
